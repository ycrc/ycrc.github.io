## November 2025


### Bouchet Expansion

As part of our annual investment, YCRC has ordered additional compute nodes for Bouchet. The majority of capital expenditures are again put towards GPU nodes, meeting the demand for accelerated computing.

- **CPU Expansion:** 26 nodes (128 CPUs each) will add 3,300+ cores to the `day` and `week` partitions, representing a 50% increase in the available cores in these partitions.
- **GPU Upgrades:** We're adding 64 RTX6000 Pro Blackwell (Server Edition) GPUs (96GB VRAM, 8 cards per node), expanding the `gpu` and `gpu_devel` partitions from 48 to 112 cards.
- **AI Initiative:** Through the AI Initiative, we're acquiring 56 flagship B200 GPUs (193GB VRAM, 8 cards per node) with dedicated high-speed networking between cards, enabling training of large foundation models on YCRC infrastructure.
- **AMD Architecture:** All new nodes will utilize AMD chips for the first time at YCRC. These chips offer higher core-density while outperforming their Intel competitors.

We are excited for these new resources and hope to have them available in Spring 2026.

### Ollama Improvements

The Ollama module has been upgraded to make running large language models on Yale systems faster, simpler, and more reliable.

- **Simpler startup:** `ollama serve`
  - Automatically starts the Ollama server on a free port
  - Automatically assigns `OLLAMA_HOST` to allow access to the running server
- **GPU detection**
  - Detects presence of GPUs and warns of slow LLM usage if not on a GPU
  - Provides an example command to request GPUs
- **Collision prevention**
  - A free port on the node is automatically assigned on launch (per unique resource allocation)
  - Attempts to launch a new server on the same compute allocation will point to the currently running server
- **Interface consistency**
  - Ollama commands are consistent across interactive, batch, and Jupyter notebooks
- **Additional commands:**
  - `ollama-host` — prints the host address of the running Ollama server
  - `ollama-down` — cleanly shuts down the server
  - To access the full list of commands after loading the module (`module load ollama/0.93`), type `ollama --help`

### Utilization of H200 GPUs on Bouchet

We are delighted to share that the H200 GPUs on Bouchet have demonstrated exceptional utilization in their first three months of operation. The 80-card deployment has maintained a 93.4% allocation rate, supporting 174 researchers across the university. To date, over 160,000 GPU-hours have been used, equivalent to $1.5M+ if spent on AWS cloud instances.

These GPUs have already made notable contributions to scientific research at the university, significantly expanding the availability and accessibility of accelerated computing.
